# Changelog

All notable changes to this project will be documented here.

The project follows [Semantic Versioning](https://semver.org/).

## 0.1.0 - 2026-07-22

### Added

- Controlled `ChatSurface` composition layer.
- Dynamic TanStack Virtual transcript with end anchoring and history prepends.
- AI SDK v6 `UIMessage.parts` rendering for text, reasoning, files, sources,
  data, tools, and tool approvals.
- Streamdown-powered streaming Markdown.
- IME-safe, mobile-aware, auto-growing composer.
- RTL-friendly logical layout and `dir="auto"` handling.
- Hardened resource URLs and bounded generic tool output.
- Compiled Tailwind CSS with shadcn-compatible tokens and standalone defaults.
- Vite example, tests, CI, npm package smoke checks, and trusted publishing.
